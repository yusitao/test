弹出单位功能封装调用方法:
      1. 首先包含PopTableSelect.js文件.
      2. 在window.onload事件里调用popTableSelectInit函数,参数是弹出单位每行选择的列数.     
          window.onload = function (e) {
              popTableSelectInit(3);
          }
      3. 弹出单位选择的浮动框的宽度可以在id为divUnit的div标签里修改width.
      4. 该封装模块可应用于其他数量不多的字典类信息的选择中.
      5. 页面初始化时要从后台给id为unitInfo的隐藏文本框赋予待选择的字典信息,格式为"id|文本;id|文本;id|文本". 选择单位后会返回文本和ID值, 
         保存ID值的文本框在实际运用中可隐藏.
